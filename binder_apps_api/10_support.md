
## `sendInquiry`
Метод доступен публично, но проверяется по CORS, чтобы доступен был только с разрешенных источников.

POST https://bapi88.apitter.com/api/v1/support/sendInquiry

{
    "name": "Иван Петров",
    "message": "Здравствуйте, у меня проблема...",
    "phone": "+79123456789",
    "inquiry_source": "site"
}

`inquiry_source` - допустимые значения: 'site' (сайт), 'app' (личный кабинет)

Дополнительно может быть передан 'submitted_from'
Тут передавать место, из которого отправлено сообщение, если на сайте несколько мест, откуда можно отправить сообщение.
Просто для стаистики, откуда в основном отправляют.

`email` - опционально можно передать почту клиента, либо тоько почту, если телефон не передается
